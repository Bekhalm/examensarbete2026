En lokal prototyp av ett webbaserat övervakningsverktyg som kontrollerar om externa webbsidor ändras och skickar notifiering (webbläsarnotis + pling) vid förändring. Systemet bygger på polling + hashing av extraherat textinnehåll.

## Förutsättningar
- Node.js 
- npm 


FK, SQLite används som lokal databasfil via Node.

## Installation

1. npm install
1. npm start
